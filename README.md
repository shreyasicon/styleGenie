# StyleGenie – AI-Powered Style Assistant

**StyleGenie** is an AI-powered fashion companion that helps users create stylish daily outfits effortlessly. By uploading photos of their wardrobe items, users receive smart outfit recommendations with complementary colors and styles, making dressing up both fun and effortless.

---

## Key Features
- **Upload Garments:** Users can add photos of their clothing items from their wardrobe.  
- **AI Detection:** Automatically identifies garment type, color, and style tags using AI (Clarifai / API4AI).  
- **Outfit Suggestions:** Generates 1–2 outfit combinations based on color harmony and style compatibility.  
- **Save Looks:** Allows users to save and revisit favorite outfit combinations.  
- **Interactive UI:** Clean, mobile-first design with smooth animations and color swatches for an engaging experience.  

---

## Unique Value
- **Personalized Wardrobe:** Uses user-uploaded items rather than generic templates, making recommendations highly relevant.  
- **Smart Color & Style Matching:** Combines AI detection with simple style logic to provide visually appealing outfit suggestions.  
- **Privacy-Focused:** User photos are processed locally or securely via serverless AI, with no permanent storage required unless saved by the user.  
- **Competition-Ready:** Polished UI, AI-driven recommendations, and social/environmental relevance make it stand out in hackathons and competitions.  

---

## Usage
1. Upload a garment photo from your wardrobe.  
2. Click **Detect** to let the AI identify the item type, color, and style.  
3. View the suggested outfit combinations curated for that item.  
4. Save your favorite looks to **My Looks** for easy reference.  
5. Repeat with additional garments to build a personalized wardrobe collection.
